引言

程序员经常在搜索源代码的时候使用关键词搜索，然而，搜索结果的准确与否依赖于程序员对于所需要的代码的功能提取关键词的能力，此外，往往还有过滤很多不符合要求的结果。更多的基于语义的搜索可以解决这些限制，但现有的方法要么不够灵活，无法匹配近义词或者复杂的查询。本文提出了一种新的方法—基于输入输出的源代码搜索，这是一种利用语义来辅助搜索的方法。
